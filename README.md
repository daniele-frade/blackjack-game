<h1>Blackjack Game App</h1>

<h3>Goals:</h3>

Implementing Blackjack game with Vanilla Javascript + HTML/CSS

<br>


<h3>Link:</h3>

https://daniele-frade.github.io/blackjack-game/

<br>


<h3>Coding languages:</h3>

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>Javascript</li>

</ul>

<br>

<ul>

</ul>
